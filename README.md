# Taskify

프론트엔드 19기 중급 프로젝트 Taskify 레포지토리

## 프로젝트 설치 & 실행

1. 레포지토리 클론

작업할 디렉토리로 이동한 뒤 아래 명령 실행:

```
git clone https://github.com/FE19-team2/Taskify.git
cd Taskify
```

2. 패키지 설치

```
pnpm install
```

설치가 끝나면 Husky(커밋 검사), lint-staged 등이 자동으로 활성화됩니다.

3. 개발 서버 실행

```
pnpm dev
```

작업 시작 전 브랜치 전략을 반드시 확인해주세요.  
[브랜치 전략](https://www.notion.so/2ae512300bdc8059a2bffce7c801c0fc)

## 노션 문서 작업 시 규칙

- 문서 작업은 Issue 생성 → 바로 노션 작업 방식으로 진행
  (문서 작업 자체는 브랜치 필요 없음)

- 작업 완료 후 해당 Issue에 노션 링크 추가

- 문제없으면 Close Issue로 종료
